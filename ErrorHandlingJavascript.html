Error Handling in Javascript - try, catch, throw & finally!

So using these four keywords we can test code and get run time errors not parse time errors which occur when code gets parsed e.g '{' (open bracket) with no '}' (closing bracket).
(Check the code for the example)

Try - is used to try piece of code

Catch - is used to catch the error

Throw - is then used to throw a custom error message or whatever

Finally - is piece of code which gets executed after all of this no matter what
